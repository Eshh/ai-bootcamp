What is Predictive Analytics?

Predictive analytics is the process of using data, statistical algorithms, and machine learning techniques to analyze
current and historical data to make predictions about future events or behaviors.

Turns historical patterns into future insights

Types of Analytics

Descriptive Analytics
Analyzes historical data to understand changes that have occurred

Provides insight into past performance and reasons behind success or failure

Uses Business Intelligence (BI), dashboards, and data visualization

Example: Monthly sales reports showing historical sales trends

Predictive Analytics
Uses statistical models and forecast techniques to predict outcomes

Forecasts future probabilities and trends based on historical data

Employs statistical algorithms and machine learning models

Example: Predicting customer churn based on engagement metrics

Prescriptive Analytics
Goes beyond predicting — suggests actions to benefit from predictions

Advises on possible outcomes and recommends actions

Uses simulation algorithms, optimization, and real-time decision-making

Example: Logistics company optimizing routes to minimize costs

How Do We Make Predictions?

Selecting Benchmark Models
Mean/Median Based Models — Use central tendency for baseline predictions

Rule Based Models — Create prediction rules from data analysis and domain knowledge

Machine Learning Models — Algorithms that learn patterns from data automatically

Rule Based Models

Creating a set of rules to make predictions based on detailed data analysis and business understanding. Rules are
manually crafted by domain experts.

Example: If Weather=Sunny & Humidity=Normal → Play=Yes

Rule Based System vs Machine Learning
Rule Based System

Machine Learning

Data → Human Analysis → Rules

Data → Algorithm → Rules

Humans craft prediction rules

Machine learns rules automatically

Works well for simple problems

Excels at complex problems

Limited by human understanding

Scales with more data

Supervised Machine Learning

In supervised learning, there is a dependent or Target Variable, and the goal is to predict the target variable based on
input or independent variables.

The algorithm learns from labeled training data

Supervised ML: Two Categories
Classification

Regression

Target variable is categorical

Target variable is continuous

Predict spam email

House price prediction

Predict movie genre

Customer lifetime value

Predict customer churn

Predict future temperature

Unsupervised Machine Learning

No Target Variable to predict. It focuses on finding hidden patterns and relationships within the data and grouping
observations with similar patterns.

Discovers structure in data without predefined answers

Types of Unsupervised Learning
Clustering — Organize similar objects into groups

Association Rules — Find interesting relations among features

Dimensionality Reduction — Reduce features while retaining important information

Reinforcement Machine Learning

Algorithms make decisions based on feedback from the environment. They follow a trial and error approach —
receiving rewards for desirable actions and penalties for undesirable ones.

Feedback Loop: Action → Environment → Reward/Penalty → Learn
